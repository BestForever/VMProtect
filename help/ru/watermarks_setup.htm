Настройка водяных знаков

Вкладка "Настройка" диалога "Водяные знаки" служит для управления водяными знаками:

Для добавления водяного знака служит одноименная кнопка. При этом откроется диалоговое окно:

  • Название - название водяного знака;
  • Значение - значение водяного знака. При помощи кнопки "Сгенерировать" можно задать случайное значение водяного знака. Поле "Значение" состоит из двух столбцов: в левом столбце можно ввести значение в Hex режиме, в правом столбце - в виде текста. При вставке водяного знака в защищаемый файл позиции, имеющие символ "?", будут иметь случайные значения.
