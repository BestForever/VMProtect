Меню "Правка"


  • Отменить - отменить последние исправления;
  • Повторить - восстановить последние отмененные исправления;
  • Вырезать - переместить выделенные данные в буфер обмена;
  • Копировать - скопировать выделенные данные в буфер обмена;
  • Вставить - вставить из буфера обмена;
  • Удалить - удалить выделенный элемент;
  • Поиск - поиск фрагмента текста

  • Перейти на адрес - перейти на заданный адрес защищаемого приложения. Возможен переход на введенный адрес либо на адрес объекта, выбранного в списке "Функции". Возможен также поиск объекта по названию;
