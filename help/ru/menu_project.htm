Меню "Проект"

  • Добавить функцию - добавить в список защищаемых объектов новую функцию. Можно задать как точный адрес функции, так и найти нужную функцию поиском по ее имени:

  • Добавить папку - добавить папку в проект:

    Использование папок не влияет на защиту находящихся в них объектов, а служит для систематизации отображения защищаемых объектов в дереве проекта. Перемещать объекты в папку можно простым перетаскиванием (drag-n-drop);

  • Добавить лицензию - добавить лицензию в проект:

    Добавляет новую лицензию пользователя и генерирует лицензионный ключ на основе пользовательских данных: имя покупателя, e-mail, дата, номер заказа и других. Внутрь ключа может быть записана как указанная информация, так и дополнительные ограничения функциональности лицензии: например, число запусков или конечная дата действия ключа.
    Для добавления лицензии необходимо сначала сгенерировать пару ключей:

  • Экспортировать ключ - экспортирует ключи проекта в выбранный формат генератора ключей:

  • Импортировать - добавляет к проекту лицензии, импортированные из серийного номера, либо из другого проекта.
  • Компилировать - компилировать защищаемое приложение с текущими параметрами защиты. Для компиляции проекта также можно нажать кнопку на панели инструментов.
  • Выполнить - запустить защищаемое приложение. Исходный/защищенный исполняемый файл приложения также можно запустить нажатием кнопки на панели инструментов. Нажатие на стрелочку позволяет задать параметры командной строки.
